# Universal EMR (Beta Web Version)

This is the Cloud version of the Universal EMR Clinic App.

**How to use:**
1. Click the Streamlit Link.
2. Enter patient details to test the interface.
3. Generate PDF prescriptions.

**⚠️ IMPORTANT:**
This is a DEMO version running on the cloud.
Patient data is NOT saved permanently. If you close the tab, the data disappears.
